# VENDETTA


# 概要

ゲームリンクスで制作した「VENDETTA」のソースコードになります。参考・使用はご自由に。
ただし、内部のテキスト(当方にてライティングしたシナリオ)に関してはこの限りではありません。
もし、無断でシナリオを使用していることが判明しましたら、相応の措置を取らせていただきます。


# 開発環境
ティラノスクリプト　ver456
Visual Studio Code ver 1.14.2
